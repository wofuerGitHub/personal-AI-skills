# Changelog

## v0.2.0 — 2026-05-09

### Added

- Provided-source mode: no substitute or supplemental sources unless authorized.
- Annual-anchor rule.
- Later-interim-only bridge rule.
- Consolidated fundamentals matrix default.
- Quantified trend column requirement.
- `*` marker for interim / non-12-month period-flow values.
- Sector/business-model metric packs.
- Testing workflow for multiple companies.

### Changed

- Default output order is now Filing Context → Consolidated Fundamentals Matrix → Executive Trend Readout → Risks & Caveats.
- Executive Summary and Trend Readout are merged.
- Quarterly reports are de-emphasized and used only when they bridge beyond the latest annual report.
- Trend commentary must include numbers.

### Deprecated

- Separate annual and quarterly tables by default.
- Vague trend language without CAGR, YoY %, bridge %, or bps.
- Searching for alternative sources in provided-source mode.
