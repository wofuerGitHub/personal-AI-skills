# shareholder-capital-preservation-score

A reusable analytical skill for scoring a company's ability to preserve shareholder capital on a 0-to-10 scale.

## Files

- `SKILL.md` — main skill definition, scoring rubric, output format, evidence rules, and safety boundaries.
- `examples.md` — worked examples and response patterns.
- `test-cases.yaml` — behavioral test cases for expected outputs.
- `schema.json` — optional structured output schema for implementations that need JSON.

## Intended Use

Use this package when assessing downside protection, permanent capital impairment risk, capital allocation quality, dilution, governance, and financial resilience.

This package is not investment advice. It is a framework for structured analysis.
